# AI and Digital Banking Talk

This repository contains speaking materials for a talk on **AI and digital banking**.

## Files
- `ai-digital-banking-talk.md` — expanded in-depth talk notes
- `ai-digital-banking-handout.html` — printable handout source
- `ai-digital-banking-ppt-outline.txt` — slide-by-slide outline

## How to use
- Read the Markdown file for the full speaking guide
- Open the HTML file in a browser and use **Print → Save as PDF** to create a PDF handout
- Use the PPT outline to build slides in PowerPoint, Google Slides, or Keynote
